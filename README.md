## 🚀 Getting Started

### Prerequisites

Install [R](https://cran.r-project.org/) and optionally [RStudio](https://posit.co/download/rstudio-desktop/).

### Install Required Packages

```r
install.packages(c("ggplot2", "dplyr", "tidyr", "stringr", "readxl", "writexl"))
```

### Run any script

```bash
Rscript 01_introduction.R
```

Or open any `.R` file in RStudio and run it interactively.
